2 programmes à lancer, l'un dans "MaxMSP Runtime" pour l'analyse audio, l'autre dans "Processing" pour les visuels. 

------------------- 0 : installation
plus de libraririe à instller normalement. 

------------------- I : démarrage
MaxMSP Runtime :
0.Lancer MaxMSP Runtime qui doit se trouver dans le dossier Applications/Max5/
1.Ouvrir le patch AnalyseAudio.maxpat
2.Choisir la carte son à utiliser, soit celle à l'intérieur du mac, soit une carte externe reliée (à brancher avant le lancement de Max pour qu'elle apparaisse dans le menu).
3.Cliquer sur le bouton sous "Ouvrir" : ce patch se ferme et un autre patch "listen" doit apparaitre.

Processing :
0.Lancer Processing, dans le dossier Applications.
1.Ouvrir le fichier ProcessingSketches/bestioles/bestioles.pde
2.Cliquer sur la flèche (Run) en haut à gauche de la fenètre de Processing. Une fenetre noire apparait.

------------------- II : utilisation avec des fichiers audio pré-enregistrés

Dans le patch "listen" de Max : 
0.Cliquer sur le haut parleur en haut de la fenètre pour mettre en route la carte son. L'icaone est verte quand la carte son est allumée.
1.Dans le panneau 1-Voix, choisir Fichier dans le menu. 
2.Cliquer sur Dossier sons pour choisir le répertoire contenant les fichiers sons pré-enregisrés (en .wav). Noramlement, les sons présents dans ce dossier doivent apparaitre dans le menu déroulant.
3.Choisir un des sons dans le menu déroulant.
4.Cliquer sur calibr. (calibration)
5.Choisir un mapping (pour l'instant seulement le n°2)
6.Cliquer sur play, et ajuster le volume de sortie si besoin.

Normalement, des bestioles apparaissent dans la fenêtre Processing. 

Répéter les étapes 1-4 et 6 pour les Percussions.

------------------- III : utilisation en live

Trop facile, il faut juste sélectionner Micro au lieu de Fichier. Et baisser les volumes de sortie pour éviter les larsens. 

------------------- IV : raccourcis clavier dans Processing

n : cree une nouvelle bestiole simple n'importe où sur l'écran
c : cree une nouvelle bestiole simple au centre
k : efface tout
r : invente une nouvelle bestiole n'importe où sur l'écran
